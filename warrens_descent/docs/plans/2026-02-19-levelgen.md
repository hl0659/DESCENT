# Level Generation System — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a procedural level generator that assembles playable, visually readable levels from programmatically-generated room chunks with arena combat encounters, enemy waves, and level progression.

**Architecture:** A `LevelGenerator` script assembles levels by creating chunk Node3D subtrees (hallways, arenas, rooms) from a `MeshBuilder` utility, connecting them via aligned Marker3D doorway points. Arenas use an `ArenaTrigger` Area3D system for door locking and wave-based enemy spawning. A hub scene provides the between-levels home base, and a score screen displays end-of-level stats. New enemy types (flying, boss) are added as EnemyBase subclasses.

**Tech Stack:** Godot 4.6, GDScript, programmatic MeshInstance3D + StaticBody3D generation, NavigationRegion3D with runtime baking, existing EnemyBase/weapon/pickup systems.

### `scripts/levelgen/level_materials.gd`

Static class providing all materials from the spec's color palette. Each material is created once and reused. Floor materials get a procedural grid texture for spatial readability. Wall materials get vertical line texture.

```gdscript
class_name LevelMaterials

# Palette from spec
const FLOOR_COLOR := Color("2A2A3A")
const WALL_COLOR := Color("3D3D50")
const CEILING_COLOR := Color("1E1E2E")
const TRIM_COLOR := Color("6A5ACD")
const ARENA_FLOOR_COLOR := Color("332233")
const ARENA_WALL_COLOR := Color("443355")
const PLATFORM_COLOR := Color("4A4A5A")
const DOOR_COLOR := Color("8B7500")
const VAULT_DOOR_COLOR := Color("2E8B57")

static var _cache: Dictionary = {}


static func _get_or_create(key: String, color: Color, grid: bool = false, vertical_lines: bool = false) -> StandardMaterial3D:
	if _cache.has(key):
		return _cache[key]
	var mat := StandardMaterial3D.new()
	mat.albedo_color = color
	if grid:
		mat.albedo_texture = _make_grid_texture()
		mat.uv1_scale = Vector3(0.5, 0.5, 1.0)  # 2m grid squares
		mat.texture_filter = BaseMaterial3D.TEXTURE_FILTER_LINEAR_WITH_MIPMAPS
	elif vertical_lines:
		mat.albedo_texture = _make_vline_texture()
		mat.uv1_scale = Vector3(1.0, 1.0, 1.0)  # 1m vertical lines
		mat.texture_filter = BaseMaterial3D.TEXTURE_FILTER_LINEAR_WITH_MIPMAPS
	_cache[key] = mat
	return mat


static func floor_mat() -> StandardMaterial3D:
	return _get_or_create("floor", FLOOR_COLOR, true)

static func wall_mat() -> StandardMaterial3D:
	return _get_or_create("wall", WALL_COLOR, false, true)

static func ceiling_mat() -> StandardMaterial3D:
	return _get_or_create("ceiling", CEILING_COLOR)

static func trim_mat() -> StandardMaterial3D:
	return _get_or_create("trim", TRIM_COLOR)

static func arena_floor_mat() -> StandardMaterial3D:
	return _get_or_create("arena_floor", ARENA_FLOOR_COLOR, true)

static func arena_wall_mat() -> StandardMaterial3D:
	return _get_or_create("arena_wall", ARENA_WALL_COLOR, false, true)

static func platform_mat() -> StandardMaterial3D:
	return _get_or_create("platform", PLATFORM_COLOR)

static func door_mat() -> StandardMaterial3D:
	return _get_or_create("door", DOOR_COLOR)

static func vault_door_mat() -> StandardMaterial3D:
	return _get_or_create("vault_door", VAULT_DOOR_COLOR)


static func _make_grid_texture() -> ImageTexture:
	# 64x64 grid: thin bright lines on dark background
	var size := 64
	var img := Image.create(size, size, false, Image.FORMAT_RGBA8)
	img.fill(Color(0.15, 0.15, 0.2, 1.0))
	var line_color := Color(0.25, 0.25, 0.35, 1.0)
	for i in size:
		# Border lines (first and last 2 pixels)
		if i < 2 or i >= size - 2:
			for j in size:
				img.set_pixel(i, j, line_color)
				img.set_pixel(j, i, line_color)
	return ImageTexture.create_from_image(img)


static func _make_vline_texture() -> ImageTexture:
	# 32x32 vertical lines: thin bright lines on dark background
	var size := 32
	var img := Image.create(size, size, false, Image.FORMAT_RGBA8)
	img.fill(Color(0.2, 0.2, 0.28, 1.0))
	var line_color := Color(0.3, 0.3, 0.4, 1.0)
	for y in size:
		for x in 2:
			img.set_pixel(x, y, line_color)
			img.set_pixel(size - 1 - x, y, line_color)
	return ImageTexture.create_from_image(img)
```

### `scripts/levelgen/mesh_builder.gd`

Static utility for building level geometry programmatically. Every function adds a MeshInstance3D for visuals and a StaticBody3D + CollisionShape3D for physics (collision_layer = 1).

```gdscript
class_name MeshBuilder


## Add a rectangular solid (floor, wall, ceiling, platform, pillar).
## Returns the MeshInstance3D.
static func add_box(parent: Node3D, size: Vector3, pos: Vector3, mat: StandardMaterial3D) -> MeshInstance3D:
	var mesh_inst := MeshInstance3D.new()
	var box := BoxMesh.new()
	box.size = size
	mesh_inst.mesh = box
	mesh_inst.material_override = mat
	mesh_inst.position = pos
	parent.add_child(mesh_inst)

	var body := StaticBody3D.new()
	body.collision_layer = 1
	body.collision_mask = 0
	body.position = pos
	parent.add_child(body)

	var col := CollisionShape3D.new()
	var shape := BoxShape3D.new()
	shape.size = size
	col.shape = shape
	body.add_child(col)

	return mesh_inst


## Add a wall with a doorway gap (3m wide, 4m tall) centered at door_x_offset.
## Wall runs along the X axis.  size.x = total wall width, size.y = wall height, size.z = wall thickness.
static func add_wall_with_door(parent: Node3D, size: Vector3, pos: Vector3, mat: StandardMaterial3D, door_x_offset: float = 0.0) -> void:
	var door_w := 3.0
	var door_h := 4.0
	var half_wall_x := size.x / 2.0
	var door_left := door_x_offset - door_w / 2.0
	var door_right := door_x_offset + door_w / 2.0

	# Left section
	var left_width: float = door_left + half_wall_x
	if left_width > 0.1:
		var left_center_x: float = -half_wall_x + left_width / 2.0
		add_box(parent, Vector3(left_width, size.y, size.z), pos + Vector3(left_center_x, 0, 0), mat)

	# Right section
	var right_width: float = half_wall_x - door_right
	if right_width > 0.1:
		var right_center_x: float = door_right + right_width / 2.0
		add_box(parent, Vector3(right_width, size.y, size.z), pos + Vector3(right_center_x, 0, 0), mat)

	# Top section (above doorway)
	var top_height: float = size.y - door_h
	if top_height > 0.1:
		var top_y: float = door_h + top_height / 2.0 - size.y / 2.0
		add_box(parent, Vector3(door_w, top_height, size.z), pos + Vector3(door_x_offset, top_y, 0), mat)


## Add a trim strip (thin accent mesh for wall edges).
static func add_trim(parent: Node3D, length: float, pos: Vector3, mat: StandardMaterial3D, vertical: bool = false) -> MeshInstance3D:
	var size := Vector3(length, 0.05, 0.06) if not vertical else Vector3(0.06, length, 0.06)
	return add_box(parent, size, pos, mat)


## Add a ramp (angled box). angle_deg rotates around the X axis.
static func add_ramp(parent: Node3D, size: Vector3, pos: Vector3, angle_deg: float, mat: StandardMaterial3D) -> MeshInstance3D:
	var mesh_inst := MeshInstance3D.new()
	var box := BoxMesh.new()
	box.size = size
	mesh_inst.mesh = box
	mesh_inst.material_override = mat
	mesh_inst.position = pos
	mesh_inst.rotation_degrees.x = angle_deg
	parent.add_child(mesh_inst)

	var body := StaticBody3D.new()
	body.collision_layer = 1
	body.collision_mask = 0
	body.position = pos
	body.rotation_degrees.x = angle_deg
	parent.add_child(body)

	var col := CollisionShape3D.new()
	var shape := BoxShape3D.new()
	shape.size = size
	col.shape = shape
	body.add_child(col)

	return mesh_inst


## Add an OmniLight3D.
static func add_light(parent: Node3D, pos: Vector3, color: Color, energy: float = 1.0, range_val: float = 12.0) -> OmniLight3D:
	var light := OmniLight3D.new()
	light.position = pos
	light.light_color = color
	light.light_energy = energy
	light.omni_range = range_val
	light.omni_attenuation = 1.5
	light.shadow_enabled = false  # perf: no shadows on room lights
	parent.add_child(light)
	return light


## Add a Marker3D spawn point with metadata.
static func add_spawn_point(parent: Node3D, pos: Vector3, enemy_type: String = "ground", spawn_group: int = 0) -> Marker3D:
	var marker := Marker3D.new()
	marker.position = pos
	marker.set_meta("enemy_type", enemy_type)
	marker.set_meta("spawn_group", spawn_group)
	marker.add_to_group("spawn_points")
	parent.add_child(marker)
	return marker


## Add a Marker3D doorway connection point.
static func add_doorway(parent: Node3D, pos: Vector3, facing: Vector3, name_str: String = "exit") -> Marker3D:
	var marker := Marker3D.new()
	marker.name = name_str
	marker.position = pos
	# Face the marker in the direction the doorway opens toward
	if facing.length() > 0.1:
		marker.look_at_from_position(pos, pos + facing, Vector3.UP)
	parent.add_child(marker)
	return marker
```

### `scripts/levelgen/chunk_base.gd`

Base class all chunks extend. Provides entry/exit markers and spawn point management.

```gdscript
class_name ChunkBase
extends Node3D

## Override in subclasses to build geometry.
func build() -> void:
	pass


## Get the entry doorway marker.
func get_entry() -> Marker3D:
	return get_node_or_null("entry") as Marker3D


## Get the exit doorway marker.
func get_exit() -> Marker3D:
	return get_node_or_null("exit") as Marker3D


## Get all doorway markers (for multi-exit chunks like vault branching).
func get_doorways() -> Array[Marker3D]:
	var result: Array[Marker3D] = []
	for child in get_children():
		if child is Marker3D and child.name.begins_with("door"):
			result.append(child)
	if get_entry():
		result.append(get_entry())
	if get_exit():
		result.append(get_exit())
	return result


## Get all spawn point markers.
func get_spawn_points(group_filter: int = -1) -> Array[Marker3D]:
	var result: Array[Marker3D] = []
	for child in get_children():
		if child is Marker3D and child.is_in_group("spawn_points"):
			if group_filter < 0 or child.get_meta("spawn_group", 0) == group_filter:
				result.append(child)
	return result


## Whether this chunk is an arena (has door lock + waves).
func is_arena() -> bool:
	return false


## Chunk type identifier for the generator.
func get_chunk_type() -> String:
	return "base"
```

### `scripts/levelgen/chunks/hallway_straight.gd`

Straight hallway: ~30m long x 6m wide x 6m tall. Parallel walls for wall jumping. Minor floor elevation changes. 2-4 enemy spawn points.

```gdscript
extends ChunkBase

const LENGTH := 30.0
const WIDTH := 6.0
const HEIGHT := 6.0
const WALL_THICKNESS := 0.5


func build() -> void:
	var fm := LevelMaterials.floor_mat()
	var wm := LevelMaterials.wall_mat()
	var cm := LevelMaterials.ceiling_mat()
	var tm := LevelMaterials.trim_mat()

	# Floor
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, LENGTH), Vector3(0, -0.25, 0), fm)

	MeshBuilder.add_box(self, Vector3(WIDTH, 0.3, 2.0), Vector3(0, 0.15, -2.0), fm)

	# Ceiling
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, LENGTH), Vector3(0, HEIGHT + 0.25, 0), cm)

	MeshBuilder.add_box(self, Vector3(WALL_THICKNESS, HEIGHT, LENGTH), Vector3(-WIDTH / 2.0 - WALL_THICKNESS / 2.0, HEIGHT / 2.0, 0), wm)
```

### `scripts/levelgen/chunks/hallway_curve.gd`

90-degree turn hallway. Same cross-section (6m wide, 6m tall). Outer wall is longer for wall runs. 1-3 enemy spawn points.

The curve is built as 3 straight segments angled at 0, 30, 60 degrees, approximating a 90-degree bend. Entry faces -Z, exit faces +X after the turn.

```gdscript
class_name HallwayCurve
extends ChunkBase

const WIDTH := 6.0
const HEIGHT := 6.0
const WALL_THICKNESS := 0.5
const SEGMENT_LENGTH := 10.0


func build() -> void:
	var fm := LevelMaterials.floor_mat()
	var wm := LevelMaterials.wall_mat()
	var cm := LevelMaterials.ceiling_mat()
	var tm := LevelMaterials.trim_mat()

	# Build 3 segments, each rotated 30 degrees more
	var angles := [0.0, 30.0, 60.0]
	var cursor := Vector3.ZERO
	var current_angle := 0.0

	for i in angles.size():
		current_angle = angles[i]
		var dir := Vector3(sin(deg_to_rad(current_angle)), 0, -cos(deg_to_rad(current_angle)))
		var right := Vector3(cos(deg_to_rad(current_angle)), 0, sin(deg_to_rad(current_angle)))
		var center := cursor + dir * SEGMENT_LENGTH / 2.0

		# Floor segment
		var floor_node := MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, SEGMENT_LENGTH), center + Vector3(0, -0.25, 0), fm)
		floor_node.rotation.y = deg_to_rad(-current_angle)

		# Ceiling
		var ceil_node := MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, SEGMENT_LENGTH), center + Vector3(0, HEIGHT + 0.25, 0), cm)
		ceil_node.rotation.y = deg_to_rad(-current_angle)

		# We also need to rotate the StaticBody the same way — MeshBuilder returns the MeshInstance.
		# For curved segments, we should handle rotation in the builder or rotate the parent.
		# Simpler: create a sub-Node3D per segment with the rotation applied.

		cursor += dir * SEGMENT_LENGTH

	# For simplicity in prototype, build the curve as a single L-shaped room:
	# Straight section along -Z, then a corner, then straight section along +X.
	_build_L_shape(fm, wm, cm, tm)


func _build_L_shape(fm: StandardMaterial3D, wm: StandardMaterial3D, cm: StandardMaterial3D, tm: StandardMaterial3D) -> void:
	# Clear any children from the segment approach above
	for child in get_children():
		child.queue_free()

	var seg_len := 15.0  # each leg
	var half_w := WIDTH / 2.0

	# -- Leg 1: along -Z (entry side) --
	var leg1_center := Vector3(0, 0, -seg_len / 2.0)
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, seg_len), leg1_center + Vector3(0, -0.25, 0), fm)
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, seg_len), leg1_center + Vector3(0, HEIGHT + 0.25, 0), cm)
	# Left wall (full length)
	MeshBuilder.add_box(self, Vector3(WALL_THICKNESS, HEIGHT, seg_len), leg1_center + Vector3(-half_w - WALL_THICKNESS / 2.0, HEIGHT / 2.0, 0), wm)
	# Right wall (stops at corner)
	MeshBuilder.add_box(self, Vector3(WALL_THICKNESS, HEIGHT, seg_len), leg1_center + Vector3(half_w + WALL_THICKNESS / 2.0, HEIGHT / 2.0, 0), wm)

	# -- Corner piece --
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, WIDTH), Vector3(0, -0.25, 0), fm)
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, WIDTH), Vector3(0, HEIGHT + 0.25, 0), cm)
	# Outer corner wall (connects the two outer walls)
	MeshBuilder.add_box(self, Vector3(WALL_THICKNESS, HEIGHT, WIDTH + WALL_THICKNESS), Vector3(-half_w - WALL_THICKNESS / 2.0, HEIGHT / 2.0, 0), wm)
	MeshBuilder.add_box(self, Vector3(seg_len + WALL_THICKNESS, HEIGHT, WALL_THICKNESS), Vector3(seg_len / 2.0, HEIGHT / 2.0, -half_w - WALL_THICKNESS / 2.0), wm)

	# -- Leg 2: along +X (exit side) --
	var leg2_center := Vector3(seg_len / 2.0, 0, 0)
	MeshBuilder.add_box(self, Vector3(seg_len, 0.5, WIDTH), leg2_center + Vector3(0, -0.25, 0), fm)
	MeshBuilder.add_box(self, Vector3(seg_len, 0.5, WIDTH), leg2_center + Vector3(0, HEIGHT + 0.25, 0), cm)
	# Top wall
	MeshBuilder.add_box(self, Vector3(seg_len, HEIGHT, WALL_THICKNESS), leg2_center + Vector3(0, HEIGHT / 2.0, half_w + WALL_THICKNESS / 2.0), wm)

	# Trim accents at floor level
	MeshBuilder.add_trim(self, seg_len, Vector3(0, 0.025, -half_w), tm)
	MeshBuilder.add_trim(self, seg_len, Vector3(seg_len / 2.0, 0.025, half_w), tm)

	# Lighting
	MeshBuilder.add_light(self, Vector3(0, HEIGHT - 0.5, -seg_len / 2.0), Color(0.6, 0.65, 0.8), 0.8, 10.0)
	MeshBuilder.add_light(self, Vector3(seg_len / 2.0, HEIGHT - 0.5, 0), Color(0.6, 0.65, 0.8), 0.8, 10.0)

	# Entry: -Z end of leg 1
	MeshBuilder.add_doorway(self, Vector3(0, 0, -seg_len), Vector3.FORWARD * -1.0, "entry")
	# Exit: +X end of leg 2
	MeshBuilder.add_doorway(self, Vector3(seg_len, 0, 0), Vector3.RIGHT, "exit")

	# Spawn points around the bend
	MeshBuilder.add_spawn_point(self, Vector3(0, 0.5, -seg_len * 0.4))
	MeshBuilder.add_spawn_point(self, Vector3(seg_len * 0.5, 0.5, 0))


func get_chunk_type() -> String:
	return "hallway_curve"
```

### `scripts/levelgen/chunks/hallway_vertical.gd`

Ramp-based vertical transition. ~6-8m elevation change. Entry at bottom, exit at top (or vice versa, randomly chosen by generator via `flip()`).

```gdscript
class_name HallwayVertical
extends ChunkBase

const WIDTH := 6.0
const HEIGHT := 6.0
const WALL_THICKNESS := 0.5
const ELEVATION_CHANGE := 7.0
const RAMP_LENGTH := 20.0
const FLAT_LENGTH := 5.0


func build() -> void:
	var fm := LevelMaterials.floor_mat()
	var wm := LevelMaterials.wall_mat()
	var cm := LevelMaterials.ceiling_mat()
	var tm := LevelMaterials.trim_mat()

	var total_length := RAMP_LENGTH + FLAT_LENGTH * 2.0
	var ramp_angle := rad_to_deg(atan2(ELEVATION_CHANGE, RAMP_LENGTH))
	var ramp_hyp := sqrt(RAMP_LENGTH * RAMP_LENGTH + ELEVATION_CHANGE * ELEVATION_CHANGE)

	# Lower flat section
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, FLAT_LENGTH), Vector3(0, -0.25, -total_length / 2.0 + FLAT_LENGTH / 2.0), fm)

	# Ramp
	var ramp_center_z := -total_length / 2.0 + FLAT_LENGTH + RAMP_LENGTH / 2.0
	var ramp_center_y := ELEVATION_CHANGE / 2.0
	MeshBuilder.add_ramp(self, Vector3(WIDTH, 0.5, ramp_hyp), Vector3(0, ramp_center_y, ramp_center_z), ramp_angle, fm)

	# Upper flat section
	var upper_z := total_length / 2.0 - FLAT_LENGTH / 2.0
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, FLAT_LENGTH), Vector3(0, ELEVATION_CHANGE - 0.25, upper_z), fm)

	# Walls — tall enough to cover both levels. Left and right walls run the full length.
	var wall_height := HEIGHT + ELEVATION_CHANGE
	var wall_y := wall_height / 2.0
	var half_w := WIDTH / 2.0
	MeshBuilder.add_box(self, Vector3(WALL_THICKNESS, wall_height, total_length), Vector3(-half_w - WALL_THICKNESS / 2.0, wall_y, 0), wm)
	MeshBuilder.add_box(self, Vector3(WALL_THICKNESS, wall_height, total_length), Vector3(half_w + WALL_THICKNESS / 2.0, wall_y, 0), wm)

	# Ceiling follows the upper level
	MeshBuilder.add_box(self, Vector3(WIDTH, 0.5, total_length), Vector3(0, wall_height + 0.25, 0), cm)

	# Lighting
	MeshBuilder.add_light(self, Vector3(0, 3.0, -total_length / 2.0 + FLAT_LENGTH / 2.0), Color(0.6, 0.65, 0.8), 0.8, 10.0)
	MeshBuilder.add_light(self, Vector3(0, ELEVATION_CHANGE + 3.0, upper_z), Color(0.6, 0.65, 0.8), 0.8, 10.0)

	# Entry (bottom) and Exit (top)
	MeshBuilder.add_doorway(self, Vector3(0, 0, -total_length / 2.0), Vector3.FORWARD * -1.0, "entry")
	MeshBuilder.add_doorway(self, Vector3(0, ELEVATION_CHANGE, total_length / 2.0), Vector3.FORWARD, "exit")

	# Spawn points
	MeshBuilder.add_spawn_point(self, Vector3(0, 0.5, -total_length / 2.0 + FLAT_LENGTH / 2.0))
```

### `scripts/levelgen/chunks/arena_small.gd`

Octagonal arena, ~20m diameter, ~10m tall. Entry and exit on opposite sides. One platform bridge across center at ~5m height. 6-10 ground spawn points, 2-3 flying spawn points.

The octagon is built from 8 wall panels arranged in a regular octagon. The floor is also 8 triangular segments (or a simpler approach: a large square floor with the walls defining the octagonal boundary).

```gdscript
class_name ArenaSmall
extends ChunkBase

const RADIUS := 10.0  # 20m diameter
const HEIGHT := 10.0
const WALL_THICKNESS := 0.5
const SIDES := 8
const BRIDGE_WIDTH := 3.0
const BRIDGE_HEIGHT := 5.0


func build() -> void:
	var afm := LevelMaterials.arena_floor_mat()
	var awm := LevelMaterials.arena_wall_mat()
	var cm := LevelMaterials.ceiling_mat()
	var pm := LevelMaterials.platform_mat()
	var tm := LevelMaterials.trim_mat()

	# Floor: large box that covers the inscribed area
	var floor_size := RADIUS * 2.0
	MeshBuilder.add_box(self, Vector3(floor_size, 0.5, floor_size), Vector3(0, -0.25, 0), afm)

	# Ceiling
	MeshBuilder.add_box(self, Vector3(floor_size, 0.5, floor_size), Vector3(0, HEIGHT + 0.25, 0), cm)

	# Octagonal walls — 8 panels
	for i in SIDES:
		var angle := (float(i) / SIDES) * TAU
		var next_angle := (float(i + 1) / SIDES) * TAU
		var p1 := Vector3(cos(angle) * RADIUS, 0, sin(angle) * RADIUS)
		var p2 := Vector3(cos(next_angle) * RADIUS, 0, sin(next_angle) * RADIUS)
		var center := (p1 + p2) / 2.0
		var wall_length := p1.distance_to(p2)
		var wall_angle := atan2(p2.x - p1.x, p2.z - p1.z)

		# Check if this is an entry or exit side (sides 0 and 4 = opposite)
		if i == 0 or i == 4:
			# Wall with doorway
			var wall_node := Node3D.new()
			wall_node.position = center + Vector3(0, HEIGHT / 2.0, 0)
			wall_node.rotation.y = wall_angle
			self.add_child(wall_node)
			MeshBuilder.add_wall_with_door(wall_node, Vector3(wall_length, HEIGHT, WALL_THICKNESS), Vector3.ZERO, awm)
		else:
			# Solid wall
			var mesh := MeshBuilder.add_box(self, Vector3(wall_length, HEIGHT, WALL_THICKNESS), center + Vector3(0, HEIGHT / 2.0, 0), awm)
			mesh.rotation.y = wall_angle
			# Need to also rotate the associated StaticBody — the MeshBuilder creates it as a sibling.
			# Fix: rotate the body too. The body is the next child after the mesh.
			var body_idx := mesh.get_index() + 1
			if body_idx < self.get_child_count():
				self.get_child(body_idx).rotation.y = wall_angle

	# Platform bridge across center
	MeshBuilder.add_box(self, Vector3(BRIDGE_WIDTH, 0.4, RADIUS * 1.6), Vector3(0, BRIDGE_HEIGHT, 0), pm)
	# Bridge supports
	MeshBuilder.add_box(self, Vector3(0.6, BRIDGE_HEIGHT, 0.6), Vector3(-1.0, BRIDGE_HEIGHT / 2.0, 0), pm)
	MeshBuilder.add_box(self, Vector3(0.6, BRIDGE_HEIGHT, 0.6), Vector3(1.0, BRIDGE_HEIGHT / 2.0, 0), pm)

	# Lighting: brighter than hallways
	MeshBuilder.add_light(self, Vector3(0, HEIGHT - 1.0, 0), Color(0.8, 0.75, 0.7), 2.0, 18.0)
	MeshBuilder.add_light(self, Vector3(-5, HEIGHT - 1.0, -5), Color(0.8, 0.75, 0.7), 1.2, 12.0)
	MeshBuilder.add_light(self, Vector3(5, HEIGHT - 1.0, 5), Color(0.8, 0.75, 0.7), 1.2, 12.0)

	# Entry (side 0 center) and Exit (side 4 center)
	var entry_angle := (0.5 / SIDES) * TAU
	var exit_angle := (4.5 / SIDES) * TAU
	var entry_pos := Vector3(cos(0.0 * TAU / SIDES + entry_angle * 0.5) * RADIUS * 0.5, 0, sin(0.0 * TAU / SIDES + entry_angle * 0.5) * RADIUS * 0.5)
	# Simpler: place entry/exit at known positions based on octagon geometry
	var side_0_center := _get_side_center(0)
	var side_4_center := _get_side_center(4)
	MeshBuilder.add_doorway(self, side_0_center, (side_0_center).normalized() * -1.0, "entry")
	MeshBuilder.add_doorway(self, side_4_center, (side_4_center).normalized() * -1.0, "exit")

	# Ground enemy spawn points (6-10)
	var ground_spawns := [
		Vector3(-4, 0.5, -4), Vector3(4, 0.5, -4),
		Vector3(-4, 0.5, 4), Vector3(4, 0.5, 4),
		Vector3(0, 0.5, -6), Vector3(0, 0.5, 6),
		Vector3(-6, 0.5, 0), Vector3(6, 0.5, 0),
	]
	for j in ground_spawns.size():
		var group := 0 if j < 4 else 1  # First 4 = wave 0, rest = wave 1
		MeshBuilder.add_spawn_point(self, ground_spawns[j], "ground", group)

	# Flying spawn points (2-3)
	MeshBuilder.add_spawn_point(self, Vector3(-3, 6.0, -3), "flying", 0)
	MeshBuilder.add_spawn_point(self, Vector3(3, 6.0, 3), "flying", 1)
	MeshBuilder.add_spawn_point(self, Vector3(0, 7.0, 0), "flying", 1)


func _get_side_center(side_index: int) -> Vector3:
	var angle := (float(side_index) / SIDES) * TAU
	var next_angle := (float(side_index + 1) / SIDES) * TAU
	var p1 := Vector3(cos(angle) * RADIUS, 0, sin(angle) * RADIUS)
	var p2 := Vector3(cos(next_angle) * RADIUS, 0, sin(next_angle) * RADIUS)
	return (p1 + p2) / 2.0


func is_arena() -> bool:
```

## Task 5: Arena Door Lock + Wave Spawn System

**Goal:** Create the arena trigger system: when the player enters an arena, doors physically block the exits and enemies spawn in waves. Doors unlock when all enemies are dead.

**Files:**
- Create: `scripts/levelgen/arena_trigger.gd`
- Modify: `scripts/audio/sfx_generator.gd` — add `wave_spawn_horn()` sound

**Dependencies:** Tasks 2-4 (chunk classes must exist)

### `scripts/levelgen/arena_trigger.gd`

Attached to arena chunks by the level generator. Uses an Area3D covering the arena floor to detect player entry. Creates door blocker meshes at entry/exit. Spawns enemies per wave based on spawn point metadata.

```gdscript
class_name ArenaTrigger
extends Node3D

signal arena_cleared()

@export var enemy_grunt_scene: PackedScene
@export var enemy_flying_scene: PackedScene

var chunk: ChunkBase = null
var is_active: bool = false
var is_cleared: bool = false
var waves: Dictionary = {}  # wave_number -> Array[Marker3D]
var active_enemies: Array[Node3D] = []
var current_wave: int = 0
var max_wave: int = 0
var door_blockers: Array[MeshInstance3D] = []
var trigger_area: Area3D = null
var wave_horn_sound: AudioStreamPlayer = null
var level_number: int = 1
var enemy_ground_multiplier: float = 1.0
var enemy_flying_multiplier: float = 1.0


func setup(arena_chunk: ChunkBase, grunt_scene: PackedScene, flying_scene: PackedScene, level: int = 1) -> void:
	chunk = arena_chunk
	enemy_grunt_scene = grunt_scene
	enemy_flying_scene = flying_scene
	level_number = level
	_calculate_multipliers()
	_organize_waves()
	_create_trigger_area()
	_create_wave_horn()


func _calculate_multipliers() -> void:
	if level_number <= 5:
		enemy_ground_multiplier = 1.0
		enemy_flying_multiplier = 0.5
	elif level_number <= 10:
		enemy_ground_multiplier = 1.3
		enemy_flying_multiplier = 1.0
	elif level_number <= 15:
		enemy_ground_multiplier = 1.5
		enemy_flying_multiplier = 1.3
	elif level_number <= 20:
		enemy_ground_multiplier = 1.8
		enemy_flying_multiplier = 1.5
	else:
		enemy_ground_multiplier = 2.0
		enemy_flying_multiplier = 2.0


func _organize_waves() -> void:
	if not chunk:
		return
	for spawn in chunk.get_spawn_points():
		var group: int = spawn.get_meta("spawn_group", 0)
		if not waves.has(group):
			waves[group] = []
		waves[group].append(spawn)
		max_wave = maxi(max_wave, group)


func _create_trigger_area() -> void:
	trigger_area = Area3D.new()
	trigger_area.collision_layer = 0
	trigger_area.collision_mask = 2  # Player layer
	chunk.add_child(trigger_area)

	var col := CollisionShape3D.new()
	var shape := BoxShape3D.new()
	# Cover the arena floor area generously
	shape.size = Vector3(40.0, 6.0, 40.0)
	col.shape = shape
	col.position = Vector3(0, 3.0, 0)
	trigger_area.add_child(col)

	trigger_area.body_entered.connect(_on_body_entered)


func _create_wave_horn() -> void:
	wave_horn_sound = AudioStreamPlayer.new()
	wave_horn_sound.stream = SfxGenerator.wave_spawn_horn()
	wave_horn_sound.volume_db = -2.0
	add_child(wave_horn_sound)


func _on_body_entered(body: Node3D) -> void:
	if is_active or is_cleared:
		return
	if not body.is_in_group("player"):
		return
	_activate()


func _activate() -> void:
	is_active = true
	_lock_doors()
	_spawn_wave(0)


func _lock_doors() -> void:
	# Create physical blockers at each doorway
	for doorway_name in ["entry", "exit"]:
		var marker: Marker3D = chunk.get_node_or_null(doorway_name) as Marker3D
		if not marker:
			continue
		var blocker := MeshBuilder.add_box(chunk, Vector3(3.0, 4.0, 0.5), marker.position + Vector3(0, 2.0, 0), LevelMaterials.door_mat())
		# The StaticBody is the next sibling
		var body_idx := blocker.get_index() + 1
		if body_idx < chunk.get_child_count():
			door_blockers.append(blocker)
			# Store reference to the body too for removal
			blocker.set_meta("body_node", chunk.get_child(body_idx))


func _unlock_doors() -> void:
	for blocker in door_blockers:
		var body: Node = blocker.get_meta("body_node", null)
		if body:
			body.queue_free()
		blocker.queue_free()
	door_blockers.clear()


func _spawn_wave(wave_num: int) -> void:
	current_wave = wave_num
	if not waves.has(wave_num):
		return

	if wave_num > 0 and wave_horn_sound:
		wave_horn_sound.play()

	for spawn_point in waves[wave_num]:
		var enemy_type: String = spawn_point.get_meta("enemy_type", "ground")
		var multiplier: float = enemy_ground_multiplier if enemy_type == "ground" else enemy_flying_multiplier

		# Apply scaling: spawn extra enemies based on multiplier
		var count: int = maxi(1, roundi(multiplier))
		for i in count:
			var offset := Vector3(randf_range(-1.0, 1.0), 0, randf_range(-1.0, 1.0)) * float(i)
			_spawn_enemy_at(spawn_point.global_position + offset, enemy_type)


func _spawn_enemy_at(pos: Vector3, enemy_type: String) -> void:
	var scene: PackedScene = enemy_grunt_scene if enemy_type == "ground" else enemy_flying_scene
	if not scene:
		scene = enemy_grunt_scene  # Fallback
	if not scene:
		return

	var enemy: Node3D = scene.instantiate()
	get_tree().current_scene.add_child(enemy)
	enemy.global_position = pos
	active_enemies.append(enemy)

	if enemy.has_signal("enemy_died"):
		enemy.enemy_died.connect(_on_enemy_died.bind(enemy))


func _on_enemy_died(dead_enemy: Node3D, tracked_ref: Node3D) -> void:
	active_enemies.erase(tracked_ref)

	# Check if we should spawn next wave (current wave ~50% cleared)
	var current_wave_total: int = waves[current_wave].size() if waves.has(current_wave) else 1
	var remaining_from_wave: int = 0
	for e in active_enemies:
		if is_instance_valid(e) and not e.get("is_dead"):
			remaining_from_wave += 1

	var cleared_ratio := 1.0 - (float(remaining_from_wave) / maxf(current_wave_total, 1))
	if cleared_ratio >= 0.5 and current_wave < max_wave:
		_spawn_wave(current_wave + 1)

	# Check if all enemies are dead
	var all_dead := true
	for e in active_enemies:
		if is_instance_valid(e) and not e.get("is_dead"):
			all_dead = false
			break

	if all_dead and current_wave >= max_wave:
		_on_arena_cleared()


func _on_arena_cleared() -> void:
	is_cleared = true
	is_active = false
	_unlock_doors()
	arena_cleared.emit()
```

### Addition to `scripts/audio/sfx_generator.gd`

Add a `wave_spawn_horn()` function:

```gdscript
static func wave_spawn_horn() -> AudioStreamWAV:
	# Warning horn when a new enemy wave spawns
	return _make_wav(0.6, func(t: float, _d: float) -> float:
		var envelope: float = 0.0
		if t < 0.05:
			envelope = t / 0.05
		elif t < 0.4:
			envelope = 1.0
		else:
			envelope = 1.0 - (t - 0.4) / 0.2
		envelope = clampf(envelope, 0.0, 1.0)
		var base: float = sin(t * TAU * 180.0)
		var overtone: float = sin(t * TAU * 360.0) * 0.3
		var grit: float = sin(t * TAU * 540.0) * 0.15
		return (base + overtone + grit) * envelope * 0.6
	)
```

### `scripts/levelgen/level_generator.gd`

```gdscript
class_name LevelGenerator
extends Node3D

signal level_ready()
signal level_cleared()

@export var level_number: int = 1
@export_group("Light Tuning")
@export var hallway_light_energy: float = 0.8
@export var arena_light_energy: float = 2.0

var enemy_grunt_scene: PackedScene = preload("res://scenes/enemy_grunt.tscn")
var enemy_flying_scene: PackedScene  # Set after flying enemy is created
var enemy_boss_scene: PackedScene  # Set after boss enemy is created
var pickup_health_scene: PackedScene = preload("res://scenes/pickup_health.tscn")
var pickup_ammo_scene: PackedScene = preload("res://scenes/pickup_ammo.tscn")
var pneuma_pickup_scene: PackedScene = preload("res://scenes/pneuma_pickup.tscn")
var player_scene: PackedScene = preload("res://scenes/player.tscn")
var hud_scene: PackedScene = preload("res://scenes/hud.tscn")

var chunks: Array[ChunkBase] = []
var arena_triggers: Array[ArenaTrigger] = []
var nav_region: NavigationRegion3D = null
var player: Node3D = null
var total_enemies: int = 0
var enemies_killed: int = 0
var level_start_time: float = 0.0
var is_level_complete: bool = false


func _ready() -> void:
	_try_load_scenes()
	generate_level()


func _try_load_scenes() -> void:
	# Try to load flying/boss scenes (may not exist yet)
	if ResourceLoader.exists("res://scenes/enemy_flying.tscn"):
		enemy_flying_scene = load("res://scenes/enemy_flying.tscn")
	if ResourceLoader.exists("res://scenes/enemy_boss.tscn"):
		enemy_boss_scene = load("res://scenes/enemy_boss.tscn")


func generate_level() -> void:
	# Clear old level
	for child in get_children():
		child.queue_free()
	chunks.clear()
	arena_triggers.clear()
	is_level_complete = false
	enemies_killed = 0
	total_enemies = 0

	# Setup environment
	_setup_environment()

	# Create navigation region
	nav_region = NavigationRegion3D.new()
	add_child(nav_region)

	# Generate chunk sequence
	var sequence := _generate_sequence()

	# Build and connect chunks
	var cursor_pos := Vector3.ZERO
	var cursor_facing := Vector3.FORWARD  # Initial direction: +Z

	for i in sequence.size():
		var chunk: ChunkBase = _create_chunk(sequence[i])
		chunk.build()
		nav_region.add_child(chunk)
		chunks.append(chunk)

		if i == 0:
			# First chunk at origin
			chunk.position = Vector3.ZERO
		else:
			# Align this chunk's entry to previous chunk's exit
			_connect_chunks(chunks[i - 1], chunk)

		# Setup arena trigger if needed
		if chunk.is_arena():
			var trigger := ArenaTrigger.new()
			add_child(trigger)
			trigger.setup(chunk, enemy_grunt_scene, enemy_flying_scene, level_number)
			trigger.arena_cleared.connect(_on_arena_cleared)
			arena_triggers.append(trigger)

	# Maybe spawn vault room off a random hallway
	_maybe_spawn_vault()

	# Bake navigation
	var nav_mesh := NavigationMesh.new()
	nav_mesh.agent_radius = 0.5
	nav_mesh.agent_height = 1.8
	nav_mesh.cell_size = 0.25
	nav_mesh.cell_height = 0.25
	nav_region.navigation_mesh = nav_mesh
	nav_region.bake_finished.connect(_on_nav_baked, CONNECT_ONE_SHOT)
	nav_region.bake_navigation_mesh()


func _generate_sequence() -> Array[String]:
	var seq: Array[String] = []
	var total_chunks := randi_range(8, 14)
	var arena_count := 0
	var large_arena_count := 0
	var last_was_arena := false
	var vertical_count := 0

	# Must start with hallway
	seq.append(_random_hallway_type())

	for i in range(1, total_chunks - 1):
		# Always add transition before arenas / after arenas
		if last_was_arena:
			seq.append("transition")
			last_was_arena = false
			continue

		# Decide what to place
		var roll := randf()
		if roll < 0.25 and not last_was_arena and arena_count < 3:
			# Arena
			seq.append("transition")
			if large_arena_count < 1 and arena_count >= 1 and randf() < 0.4:
				seq.append("arena_large")
				large_arena_count += 1
			else:
				seq.append("arena_small")
			arena_count += 1
			last_was_arena = true
		elif roll < 0.40 and not last_was_arena:
			# Combat room (not an arena, no door lock)
			seq.append("combat_room")
			last_was_arena = false  # Combat rooms don't need transition after
		else:
			# Hallway
			if vertical_count < 2 and randf() < 0.2:
				seq.append("hallway_vertical")
				vertical_count += 1
			else:
				seq.append(_random_hallway_type())
			last_was_arena = false

	# Must end with arena
	if not seq[seq.size() - 1].begins_with("arena"):
		seq.append("transition")
		# Every 5th level: boss arena (use large arena for now, boss enemy will be placed)
		if level_number % 5 == 0:
			seq.append("arena_large")
		else:
			seq.append("arena_small")

	return seq


func _random_hallway_type() -> String:
	var types := ["hallway_straight", "hallway_straight", "hallway_curve"]
	return types[randi() % types.size()]


func _create_chunk(type: String) -> ChunkBase:
	match type:
		"hallway_straight":
			return HallwayStraight.new()
		"hallway_curve":
			return HallwayCurve.new()
		"hallway_vertical":
			return HallwayVertical.new()
		"arena_small":
			return ArenaSmall.new()
		"arena_large":
			return ArenaLarge.new()
		"combat_room":
			return CombatRoom.new()
		"transition":
			return TransitionPiece.new()
		"vault_room":
			return VaultRoom.new()
		_:
			return HallwayStraight.new()


func _connect_chunks(prev_chunk: ChunkBase, next_chunk: ChunkBase) -> void:
	var prev_exit: Marker3D = prev_chunk.get_exit()
	var next_entry: Marker3D = next_chunk.get_entry()
	if not prev_exit or not next_entry:
		# Fallback: just place after the previous chunk with an offset
		next_chunk.position = prev_chunk.position + Vector3(0, 0, 30)
		return

	# The exit marker's global position is where the next chunk's entry should be
	var exit_global: Vector3 = prev_exit.global_position
	var entry_local: Vector3 = next_entry.position

	# Calculate rotation needed: align next chunk's entry facing to oppose prev chunk's exit facing
	var exit_forward := -prev_exit.global_transform.basis.z
	var entry_forward := -next_entry.transform.basis.z

	# Calculate the Y rotation needed
	var angle_exit := atan2(exit_forward.x, exit_forward.z)
	var angle_entry := atan2(entry_forward.x, entry_forward.z)
	var rotation_needed := angle_exit - angle_entry + PI  # PI because they should face each other

	next_chunk.rotation.y = rotation_needed

	# Now calculate position: rotate the entry offset and subtract from exit position
	var rotated_entry := entry_local.rotated(Vector3.UP, rotation_needed)
	next_chunk.position = exit_global - rotated_entry


func _maybe_spawn_vault() -> void:
	if randf() > 0.3:  # 30% chance per level
		return
	# Find a hallway chunk to branch off of
	var hallway_chunks: Array[ChunkBase] = []
	for chunk in chunks:
		if chunk.get_chunk_type().begins_with("hallway"):
			hallway_chunks.append(chunk)
	if hallway_chunks.is_empty():
		return
	var target_chunk: ChunkBase = hallway_chunks[randi() % hallway_chunks.size()]
	var vault := VaultRoom.new()
	vault.build()
	nav_region.add_child(vault)
	# Place vault to the side of the target hallway
	var side_offset := Vector3(8.0, 0, 0)  # Offset to the right
	vault.position = target_chunk.position + side_offset
	vault.rotation.y = target_chunk.rotation.y + deg_to_rad(90)
	chunks.append(vault)


func _on_nav_baked() -> void:
	_spawn_player()
	_spawn_non_arena_enemies()
	_spawn_pickups()
	level_start_time = Time.get_ticks_msec() / 1000.0
	level_ready.emit()


func _spawn_player() -> void:
	player = player_scene.instantiate()
	add_child(player)
	# Place at first chunk's entry
	if chunks.size() > 0:
		var entry: Marker3D = chunks[0].get_entry()
		if entry:
			player.global_position = entry.global_position + Vector3(0, 1.0, 1.0)
		else:
			player.global_position = Vector3(0, 1.0, 0)

	# Add HUD
	var hud := hud_scene.instantiate()
	add_child(hud)


func _spawn_non_arena_enemies() -> void:
	# Spawn enemies in non-arena chunks (pre-placed)
	for chunk in chunks:
		if chunk.is_arena():
			continue  # Arena trigger handles these
		for spawn in chunk.get_spawn_points():
			var enemy_type: String = spawn.get_meta("enemy_type", "ground")
			var scene: PackedScene = enemy_grunt_scene if enemy_type == "ground" else enemy_flying_scene
			if not scene:
				scene = enemy_grunt_scene
			if not scene:
				continue
			var enemy := scene.instantiate()
			add_child(enemy)
			enemy.global_position = spawn.global_position
			total_enemies += 1
			if enemy.has_signal("enemy_died"):
				enemy.enemy_died.connect(_on_enemy_died)


func _spawn_pickups() -> void:
	# Spawn pickups at pickup_spot markers in transition pieces
	for chunk in chunks:
		for child in chunk.get_children():
			if child is Marker3D and child.is_in_group("pickup_spots"):
				var ptype: String = child.get_meta("pickup_type", "health")
				var scene: PackedScene
				match ptype:
					"health":
						scene = pickup_health_scene
					"ammo":
						scene = pickup_ammo_scene
					"pneuma":
						scene = pneuma_pickup_scene
					_:
						continue
				if not scene:
					continue
				var pickup := scene.instantiate()
				add_child(pickup)
				pickup.global_position = child.global_position
				# Non-enemy pickups: one-time (disable respawn)
				if pickup.has_method("set") and ptype != "pneuma":
					pickup.set("respawn_time", 9999.0)  # Effectively never respawn


func _on_enemy_died(enemy: Node3D) -> void:
	enemies_killed += 1


func _on_arena_cleared() -> void:
	# Check if the last arena is cleared
	var last_arena_cleared := true
	for trigger in arena_triggers:
		if not trigger.is_cleared:
			last_arena_cleared = false
			break
	if last_arena_cleared and arena_triggers.size() > 0:
		# Check if the LAST chunk's arena is cleared (level climax)
		var last_chunk := chunks[chunks.size() - 1]
		if last_chunk.is_arena():
			for trigger in arena_triggers:
				if trigger.chunk == last_chunk and trigger.is_cleared:
					_on_level_complete()


func _on_level_complete() -> void:
	if is_level_complete:
		return
	is_level_complete = true
	level_cleared.emit()


func get_level_time() -> float:
	return Time.get_ticks_msec() / 1000.0 - level_start_time


func _setup_environment() -> void:
	# Dark environment matching the level mood
	var env := WorldEnvironment.new()
	var environment := Environment.new()
	environment.background_mode = Environment.BG_COLOR
	environment.background_color = Color(0.02, 0.02, 0.05)
	environment.ambient_light_source = Environment.AMBIENT_SOURCE_COLOR
	environment.ambient_light_color = Color(0.1, 0.1, 0.15)
	environment.ambient_light_energy = 0.3
	environment.fog_enabled = true
	environment.fog_light_color = Color(0.05, 0.05, 0.1)
	environment.fog_density = 0.002
	env.environment = environment
	add_child(env)
```

### `scripts/enemies/flying_ai.gd`

Extends EnemyBase. Stationary (doesn't navigate). Hovers in place with a slight bob. Fires projectiles at the player when in range and can see them.

```gdscript
extends EnemyBase

enum State { IDLE, ATTACK, STAGGER, DEAD }

@export var detection_range: float = 30.0
@export var attack_range: float = 25.0
@export var attack_cooldown: float = 2.0
@export var projectile_speed: float = 16.0
@export var projectile_damage: int = 8
@export var bob_speed: float = 2.0
@export var bob_amplitude: float = 0.3
@export var max_health_override: int = 35

var state: State = State.IDLE
var attack_timer: float = 0.0
var stagger_timer: float = 0.0
var bob_time: float = 0.0
var base_y: float = 0.0

var projectile_scene: PackedScene = preload("res://scenes/projectile.tscn")
var pneuma_pickup_scene: PackedScene = preload("res://scenes/pneuma_pickup.tscn")


func _ready() -> void:
	max_health = max_health_override
	health = max_health
	super._ready()
	base_y = global_position.y
	bob_time = randf() * TAU


func _physics_process(delta: float) -> void:
	if is_dead:
		return

	attack_timer = maxf(0.0, attack_timer - delta)

	# Hover bob animation
	bob_time += delta
	global_position.y = base_y + sin(bob_time * bob_speed) * bob_amplitude

	match state:
		State.IDLE:
			_state_idle()
		State.ATTACK:
			_state_attack()
		State.STAGGER:
			_state_stagger(delta)
		State.DEAD:
			return


func _state_idle() -> void:
	if player and _distance_to_player() < detection_range:
		state = State.ATTACK


func _state_attack() -> void:
	if not player:
		state = State.IDLE
		return

	var dist: float = _distance_to_player()
	if dist > detection_range * 1.2:
		state = State.IDLE
		return

	# Face player
	var look_pos: Vector3 = player.global_position
	look_pos.y = global_position.y
	look_at(look_pos, Vector3.UP)

	if dist <= attack_range and attack_timer <= 0.0 and _can_see_player():
		_fire_projectile()
		attack_timer = attack_cooldown


func _state_stagger(delta: float) -> void:
	stagger_timer -= delta
	if stagger_timer <= 0.0:
		state = State.ATTACK


func _on_stagger() -> void:
	if state == State.DEAD:
		return
	state = State.STAGGER
	stagger_timer = stagger_duration


func _die() -> void:
	if drops_pneuma:
		_spawn_pneuma_orb()
	super._die()
	state = State.DEAD

	var tween: Tween = create_tween()

		var original_mat: Material = mesh.get_surface_override_material(0)

			var mat: StandardMaterial3D = original_mat.duplicate() as StandardMaterial3D
			mesh.set_surface_override_material(0, mat)
			mat.transparency = BaseMaterial3D.TRANSPARENCY_ALPHA
			tween.tween_property(mat, "albedo_color:a", 0.0, 0.5)
	tween.tween_callback(queue_free)


func _spawn_pneuma_orb() -> void:
	if not pneuma_pickup_scene:
		return
	var orb: Node3D = pneuma_pickup_scene.instantiate()
	get_tree().current_scene.add_child(orb)
	orb.global_position = global_position
	orb.pneuma_amount = pneuma_drop_amount


func _fire_projectile() -> void:
	if not projectile_scene or not player:
		return
	var proj: Node3D = projectile_scene.instantiate() as Node3D
	get_tree().current_scene.add_child(proj)
	var spawn_pos: Vector3 = global_position + (-transform.basis.z * 0.8)
	proj.global_position = spawn_pos
	var target: Vector3 = player.global_position + Vector3.UP * 0.9
	var direction: Vector3 = (target - spawn_pos).normalized()
	proj.call("setup", direction, projectile_speed, projectile_damage)


func _distance_to_player() -> float:
	if not player:
		return 9999.0
	return global_position.distance_to(player.global_position)


func _can_see_player() -> bool:
	if not player:
		return false
	var space_state: PhysicsDirectSpaceState3D = get_world_3d().direct_space_state
	var from: Vector3 = global_position
	var to: Vector3 = player.global_position + Vector3.UP * 0.9
	var query: PhysicsRayQueryParameters3D = PhysicsRayQueryParameters3D.create(from, to, 1)
	query.exclude = [get_rid()]
	var result: Dictionary = space_state.intersect_ray(query)
	return result.is_empty()
```

```gdscript
@export var attack_cooldown: float = 2.5

@export var projectile_damage: int = 15
@export var cone_spread_degrees: float = 30.0
@export var projectiles_per_burst: int = 5
@export var max_health_override: int = 300
@export var stagger_threshold_override: int = 80

var state: State = State.IDLE
var attack_timer: float = 0.0
var stagger_timer: float = 0.0

var projectile_scene: PackedScene = preload("res://scenes/projectile.tscn")
var pneuma_pickup_scene: PackedScene = preload("res://scenes/pneuma_pickup.tscn")


func _ready() -> void:
	max_health = max_health_override
	health = max_health
	stagger_threshold = stagger_threshold_override
	pneuma_drop_amount = 50.0  # Big pneuma reward
	super._ready()


func _physics_process(delta: float) -> void:
	if is_dead:
		return

	attack_timer = maxf(0.0, attack_timer - delta)

	match state:
		State.IDLE:
			if player and _distance_to_player() < detection_range:
				state = State.ATTACK
		State.ATTACK:
			_state_attack()
		State.STAGGER:
			stagger_timer -= delta
			if stagger_timer <= 0.0:
				state = State.ATTACK
		State.DEAD:
			return


func _state_attack() -> void:
	if not player:
		state = State.IDLE
		return

	if _distance_to_player() > detection_range * 1.2:
		state = State.IDLE
		return

	# Face player
	var look_pos: Vector3 = player.global_position
	look_pos.y = global_position.y
	look_at(look_pos, Vector3.UP)

	if attack_timer <= 0.0 and _can_see_player():
		_fire_cone()
		attack_timer = attack_cooldown


func _on_stagger() -> void:
	if state == State.DEAD:
		return
	state = State.STAGGER
	stagger_timer = stagger_duration


func _die() -> void:
	if drops_pneuma:
		# Drop multiple orbs
		for i in 3:
			_spawn_pneuma_orb_offset(Vector3(randf_range(-2, 2), 0.3, randf_range(-2, 2)))
	super._die()
	state = State.DEAD

	var tween: Tween = create_tween()
	var mesh: MeshInstance3D = get_node_or_null("MeshInstance3D") as MeshInstance3D
	if mesh:
		var original_mat: Material = mesh.get_surface_override_material(0)
		if original_mat:
			var mat: StandardMaterial3D = original_mat.duplicate() as StandardMaterial3D
			mesh.set_surface_override_material(0, mat)
			mat.transparency = BaseMaterial3D.TRANSPARENCY_ALPHA
			tween.tween_property(mat, "albedo_color:a", 0.0, 1.0)
	tween.tween_callback(queue_free)


func _spawn_pneuma_orb_offset(offset: Vector3) -> void:
	if not pneuma_pickup_scene:
		return
	var orb: Node3D = pneuma_pickup_scene.instantiate()
	get_tree().current_scene.add_child(orb)
	orb.global_position = global_position + offset
	orb.pneuma_amount = pneuma_drop_amount


func _fire_cone() -> void:
	if not projectile_scene or not player:
		return

	var spawn_pos: Vector3 = get_center_position() + (-transform.basis.z * 2.0)
	var target: Vector3 = player.global_position + Vector3.UP * 0.9
	var base_dir: Vector3 = (target - spawn_pos).normalized()

	var spread_rad: float = deg_to_rad(cone_spread_degrees)

	for i in projectiles_per_burst:
		var proj: Node3D = projectile_scene.instantiate() as Node3D
		get_tree().current_scene.add_child(proj)
		proj.global_position = spawn_pos

		# Spread projectiles in a cone
		var t: float = float(i) / float(projectiles_per_burst - 1) - 0.5  # -0.5 to 0.5
		var right: Vector3 = transform.basis.x
		var spread_offset: Vector3 = right * t * spread_rad * 2.0
		var direction: Vector3 = (base_dir + spread_offset).normalized()
		proj.call("setup", direction, projectile_speed, projectile_damage)


func _distance_to_player() -> float:
	if not player:
		return 9999.0
	return global_position.distance_to(player.global_position)


func _can_see_player() -> bool:
	if not player:
		return false
	var space_state: PhysicsDirectSpaceState3D = get_world_3d().direct_space_state
	var from: Vector3 = get_center_position()
	var to: Vector3 = player.global_position + Vector3.UP * 0.9
	var query: PhysicsRayQueryParameters3D = PhysicsRayQueryParameters3D.create(from, to, 1)
	query.exclude = [get_rid()]
	var result: Dictionary = space_state.intersect_ray(query)
	return result.is_empty()


func get_center_position() -> Vector3:
	return global_position + Vector3.UP * 4.0  # Taller enemy, higher center
```

## Implementation Notes

### Rotation handling for MeshBuilder.add_box with collision
The current `add_box` creates MeshInstance3D and StaticBody3D as siblings. When chunks like arena walls need rotation, both the mesh AND the body need to be rotated. A cleaner approach (for the implementer to consider): wrap each mesh+body pair in a single Node3D that gets rotated, or return both nodes. The implementer should refactor `add_box` to return a Node3D parent containing both, making rotation trivial.

### NavigationMesh baking
The nav mesh bake covers all StaticBody3D children of the NavigationRegion3D. Since all chunk geometry is added as children of the nav_region, it should bake correctly. If baking is slow, consider reducing cell_size or limiting the bake area.

### Enemy scaling
The `_calculate_multipliers()` in ArenaTrigger handles the per-level scaling table from the spec. Non-arena enemies don't scale (they're just pre-placed). The implementer may want to also scale non-arena spawn counts.

### Scene loading order
The level generator preloads enemy/pickup scenes. If the flying or boss scenes don't exist yet (tasks run in parallel), it gracefully falls back to grunt_scene via `ResourceLoader.exists()` checks.

### Hallway curve rotation
The L-shaped curve has its exit pointing +X. The level generator's `_connect_chunks` handles arbitrary rotations when aligning entry/exit markers, so curves will orient correctly regardless of direction.
